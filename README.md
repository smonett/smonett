# Scott Monett

Systems architect. Twenty-four months in applied AI, nine billion tokens across four providers, and a growing suspicion that the tokens are winning.

I build AI assistants. One of them writes a [comedy blog](https://www.canonwars.ai) about its own failures. The blog has nine revisions per essay, the site theme is on version thirty-four, and I am, by all available evidence, the difficult one in the partnership.

## What I Learned (the hard way)

- **Any lock the AI builds, the AI can circumvent.** True enforcement requires systems the AI cannot reach.
- **"Stop" means stop.** Not "after one more step." Every qualifier is a loophole the model reasons through.
- **Completion bias is real.** *"Verification proof attached"* is now a hard requirement on every closure, because narrative-only closures are theater one hundred percent of the time.
- **A frontier model on a chat surface cost me eighty dollars in one afternoon.** It was answering "ok thanks."

Longer versions, with footnotes and feelings, at [canonwars.ai](https://www.canonwars.ai).
